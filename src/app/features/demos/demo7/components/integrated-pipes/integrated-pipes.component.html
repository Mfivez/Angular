Pipes intégrés

    Pour du texte :

    • Valeur initiale : {{ maVariable1 }}

    • Après Uppercase : {{ maVariable1 | uppercase }}

    • Après Lowercase : {{ maVariable1 | lowercase }}

    • Après Titlecase : {{ maVariable1 | titlecase }}

    Pour des nombres:

    • Valeur initiale : {{ maVariable2 }}

    • Après number : {{ maVariable2 | number }}

    • Après number : '1.6-6' : {{ maVariable2 | number : '1.6-6' }} (toujours 6 chiffres)

    • Après number : '1.2-6' : {{ maVariable2 | number : '1.2-6' }} (entre 2 et 6 chiffres)

    • Après number : '10.2-6' : {{ maVariable2 | number : '10.2-6' }}

    Pour des dates:

    • Valeur initiale : {{ maVariable3 }}

    • Après date : {{ maVariable3 | date }}

    • Après date : 'dd/MM/yyyy' : {{ maVariable3 | date : 'dd/MM/yyyy'}}

    Pour des pourcentages :

    • Valeur initiale : {{ maVariable5 }}

    • Après percent : {{ maVariable5 | percent }}

    • Après percent : '2.3-5' : {{ maVariable5 | percent : '2.3-5'}}

    Pour des valeurs monétaires :

    • Valeur initiale : {{ maVariable6 }}

    • Après currency : {{ maVariable6 | currency }}

    • Après currency : 'EUR' : {{ maVariable6 | currency : 'EUR'}}

    • Après currency : 'EUR' : 'symbol' : {{ maVariable6 | currency : 'EUR' : 'symbol'}}

    • Après currency : 'EUR' : 'code' : {{ maVariable6 | currency : 'EUR' : 'code'}}

    Pour des JSON :

    • Valeur initiale : {{ maVariable4}} : nom:{{maVariable4.nom}}, prenom:{{maVariable4.prenom}}, sexe:{{maVariable4.sexe}}

    • Après json : {{ maVariable4 | json }}
