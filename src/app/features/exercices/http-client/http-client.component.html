Exercice - CRUD avec HttpClient

Contexte :

Créer sur cette Url, une mini-app qui interagit avec une API RESTful en effectuant des opérations CRUD. Nous simulerons un système de gestion des utilisateurs où tu pourras :

  • Récupérer la liste des utilisateurs (GET)

  • Ajouter un nouvel utilisateur (POST)

  • Mettre à jour un utilisateur existant (PUT)

  • Supprimer un utilisateur (DELETE)

Pour cet exercice, nous allons utiliser une API de test comme JSONPlaceholder

Objectifs :

  • Utiliser HttpClient pour effectuer des requêtes HTTP.

  • Gérer les réponses et erreurs des requêtes.

  • Manipuler les données via des formulaires et afficher les résultats à l'écran.

  • Appliquer des Observables pour gérer les flux de données asynchrones.

Enoncé :

1.Configurer le service HttpClient :

  • Crée un service Angular pour interagir avec l'API et implémenter les différentes méthodes HTTP.

2. Composant pour afficher les utilisateurs :

  • Un composant Angular pour afficher la liste des utilisateurs récupérés depuis l'API.

3. Composant pour ajouter/mettre à jour un utilisateur :

  • Ce composant contiendra un formulaire pour ajouter ou modifier un utilisateur.

4. Supprimer un utilisateur :

  • Ajoute un bouton à chaque utilisateur pour le supprimer de la liste via une requête DELETE.
