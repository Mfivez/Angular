Exercices : Input | Output

Enoncé :

Mettre en place une shopping list en 2 composants.

Dans le 1er :

  • Un champ textuel permettant l'ajout d'un article à la liste

Dans le 2nd :

  • L'affichage de la liste et la possibilité de supprimer un article.

⚠️ Les 2 composant doivent être affiché sur la même page !
