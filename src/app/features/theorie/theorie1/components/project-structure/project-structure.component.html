4. Analyse de l'arborescence initiale du projet :

  • 1. `node_modules` : Contient tous les modules nécessaires au framework

  • 2. `src` : Contient le code de notre projet

  • 3. `angular.json` : Configuration de l'espace de travail du projet

    • C'est ici que tu trouveras les configurations définies lors de la création du projet, comme --skip-tests ou --standalone false.

  • 4. `package.json` : Listes & versions des packages du projet

  • 5. `tsconfig.json` : Configuration de la compilation TypeScript
