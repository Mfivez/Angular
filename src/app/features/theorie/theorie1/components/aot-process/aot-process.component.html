3. Le processus Ahead-of-time (AOT) :

Technique de compilation utilisée par Angular pour améliorer les performances des applications.

L'AOT compile l'application avant son exécution, convertissant le code Angular en JavaScript optimisé avant que le navigateur ne le charge. Cela rend le chargement initial plus rapide et réduit la taille des fichiers transmis au navigateur.

C'est aussi ce processus qui permet à l'application de se recharger automatiquement à chaque modification sans devoir relancer le serveur.
