Guards - Protéger les routes

Les Guards sont des fonctions spéciales qui contrôlent l'accès à certaines routes. Ils permettent de décider si un utilisateur peut ou non accéder à une page.

Types de Guards :

  • canActivate : Vérifie si un utilisateur a le droit d'accéder à la page (par exemple, vérifier s'il est connecté).
  • canDeactivate : Vérifie si un utilisateur peut quitter la page (par exemple, s'il n'a pas encore sauvegardé des modifications).
