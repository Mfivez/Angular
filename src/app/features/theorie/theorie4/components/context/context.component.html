Contexte :

Dans le cadre du développement de notre application Angular, nous avons déjà permis de naviguer entre différentes pages en modifiant manuellement l'URL dans la barre de recherche du navigateur. Cependant, cette méthode n'est pas idéale en termes d'ergonomie pour l'utilisateur. Pour améliorer cela, nous allons intégrer une barre de navigation qui permet un accès facile et rapide à différentes sections de l'application.
