Qu'est-ce que SessionStorage ?

Le SessionStorage est similaire à LocalStorage, mais ses données sont limitées à la session de navigation actuelle. Cela signifie que les données stockées sont supprimées lorsque l'utilisateur ferme l'onglet ou le navigateur.

Caractéristiques :

  • Capacité de stockage : environ 5MB selon le navigateur.

  • Les données sont accessibles seulement durant la session active.

  • Accessible via JavaScript à travers sessionStorage.



Comment utiliser SessionStorage ?

Les méthodes pour SessionStorage sont identiques à celles du LocalStorage :

  • setItem('clé', 'valeur') : Ajouter ou mettre à jour une entrée.

  • getItem('clé') : Récupérer la valeur stockée pour une clé donnée.

  • removeItem('clé') : Supprimer une entrée spécifique.
