Qu'est-ce que LocalStorage ?

Le LocalStorage est un espace de stockage persistant côté client qui permet de conserver des données même après que l'utilisateur ait fermé l'onglet ou redémarré son navigateur.

Les données restent disponibles jusqu'à ce qu'elles soient supprimées manuellement ou par du code. Chaque application web peut y accéder pour conserver des informations entre différentes sessions de navigation.

Caractéristiques :

  • Capacité de stockage : environ 5MB selon le navigateur.

  • Les données sont stockées indéfiniment, à moins d'être supprimées par le code ou manuellement par l'utilisateur.

  • Accessible via JavaScript à travers localStorage.



Comment utiliser LocalStorage ?

Pour utiliser LocalStorage, on peut utiliser trois méthodes principales :

  • setItem('clé', 'valeur') : Ajouter ou mettre à jour une entrée.

  • getItem('clé') : Récupérer la valeur stockée pour une clé donnée.

  • removeItem('clé') : Supprimer une entrée spécifique.
